# CoreLight Cyberpunk Dashboard

Système RGB audio-réactif Arduino – WS2812B, ventilateurs synchronisés, ambiance cyberpunk.

## Fichiers du projet

- `index.html` — Page d'accueil avec animation tempête foudre
- `dashboard.html` — Dashboard avec onglets, LED strip interactif, hardware, journal
- `script.js` — Logique partagée (onglets, journal, thème, typewriter)
- `storm-anim.js` — Animation tempête de fond pour `index.html`
- `style.css` — Styles principaux (thème néon cyberpunk)
- `logo.png` — Logo CoreLight
- `prototype1.jpg` — Photo prototype 1
- `prototype2.jpg` — Photo prototype 2
- `tempete_foudre.py` — Script Python générateur du GIF tempête (référence)

## Déploiement GitHub Pages

1. Place tous les fichiers à la racine du dépôt (pas dans un sous-dossier).
2. Va dans **Settings > Pages** > choisis la branche `main` et le dossier `/` (racine).
3. Accède à ton site sur `https://<ton-utilisateur>.github.io/<nom-du-repo>/`.

Les chemins sont relatifs — tout fonctionnera directement.
