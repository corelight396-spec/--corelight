# CoreLight Cyberpunk Dashboard

Ce projet est prêt pour GitHub Pages. Place tous ces fichiers à la racine de ton dépôt GitHub (pas dans un sous-dossier) :

- index.html
- dashboard.html
- script.js
- style.css
- bg-anim.js
- audio-visualizer.js
- logo.png
- icons/ (dossier)

## Conseils GitHub Pages
- Va dans les paramètres du dépôt > Pages > choisis la branche `main` et le dossier `/` (racine).
- Accède à ton site sur `https://<ton-utilisateur>.github.io/<nom-du-repo>/`.
- Les chemins des fichiers sont déjà relatifs, donc tout fonctionnera.
